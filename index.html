Web Scrappers 🖧

Durante o período de um ano trabalhei no desenvolvimento de tecnologias de Scrapping, como desenvolvedor lead, conduzindo a criação de uma robusta rede de coletagem de dados, projetada para agrupar e analisar dados de mercado em larga escala, com uma abordagem que prioriza a discrição e a eficiência.

/*Especificações Técnicas*/

  • <Linguagens e Frameworks> C#, Next.js
  • <Mascaramento de IP> Proxy rotativo, VPN, Desenvolvimento em Núvem
  • <Controle de Detecção> Remoção de indicadores de automação, headers fictícios, comportamento humano simulado

/*Impacto e Utilidade*/

Os dados coletados foram rotineramente utilizados por uma ampla gama de profissionais, oferecendo uma ferramenta que permitiu a análise precisa e em tempo real de tendências de mercado. A tecnologia continua em uso atualmente, e tem a infraestrutra organizada de modo que permita expansão das atividades.

Descrição Completa

Linguagens e Frameworks: O backend foi construído em C#, com um painel de ações desenvolvido em Next.js para facilitar a visualização e gerenciamento das operações de scraping.

Mascaramento de IP: Implementamos um sistema de proxy rotativo, hospedado em uma VPN integrada em servidores em núvem, o que permitiu a máscara de rede e acesso seguro a bancos de dados específicos. Esse setup garantiu que as operações fossem praticamente indetectáveis, permitindo uma coleta contínua e segura de informações sensíveis.

Controle de Detecção: Para evitar a detecção de bots, adotamos várias técnicas avançadas:

Remoção de Indicadores de Automação: Todas as configurações que pudessem denunciar a existência de um robô foram removidas ou alteradas.

Headers Fictícios e Dados Dinâmicos: Integração de headers dinâmicos, informações de hardware, tela e navegador simulando um ambiente de usuário real.

Simulação de Comportamento Humano: Implementamos cooldowns entre requisições, movimentos de scroll e ações dinâmicas no HTML, que imitam perfeitamente o comportamento humano.

Data-Parser ⚒

Uma solução avançada para o processamento e validação de grandes volumes de dados, utilizando técnicas de inteligência artificial para garantir a precisão e uniformidade dos dados.

/*Especificações Técnicas*/

  • <Linguagens e Ferramentas> C#, Python
  • <Validação de Dados com IA> LLMs com fine-tuning em bancos de dados
  • <Otimização e Performance> Otimizado para PostgreSQL, técnicas de sorting, modularidade

/*Impacto e Utilidade*/

Vital para o processo empregado, a ferramenta permitiu a correção e uniformização de grandes bancos de dados usando modelos de larga escala de linguagem (LLMs), em um sistema complexo de validações, para entrega precisa e rápida de dados limpos e validados.

Descrição Completa

O parser foi desenvolvido principalmente em C#, com módulos específicos em Python para validação de dados, aproveitando as capacidades de processamento de dados dessas linguagens.

Utilizamos Large Language Models (LLMs) ajustados com fine-tuning em bancos de dados específicos, o que nos permitiu criar um arsenal de referências robustas para a uniformização e validação do conteúdo. As técnicas de prompting e gerenciamento de redes neurais garantiram que os dados fossem processados de forma eficiente e precisa.

O parser foi otimizado para trabalhar com bancos de dados PostgreSQL, sendo capaz de receber, confirmar e tratar grandes volumes de dados. Utilizamos técnicas de sorting e controle de frequência de subidas e requisições para evitar delays e overflows, assegurando uma operação fluida e confiável.

O projeto foi estruturado com foco na modularidade e CleanCode, facilitando futuras manutenções e adaptações conforme as necessidades da aplicação evoluem.

Projetos Maiores Vinculados à 42SP ☆

Durante minha formação na 42SP, atuei desenvolvendo projetos desafiadores, que exigiram uma profunda compreensão de C e desenvolvimento de baixo nível.

/*Push-Swap 🖅*/

Link para o Projeto

Desenvolvido utilizando operações de bitwise, permitindo uma comunicação extremamente eficiente e precisa a nível de bits.

Descrição Completa

O Push-Swap foi desenvolvido utilizando operações de bitwise, permitindo uma comunicação extremamente eficiente e precisa a nível de bits. O algoritmo, desenvolvido por mim, foi inspirado na lógica das máquinas de socar de arcade, garantindo que os dados sejam transmitidos apenas quando a confirmação do último bit é recebida, utilizando um sistema de alavancas lógicas para calcular tempo e eficiência.

Impacto: Este projeto destacou-se pela rapidez e precisão da comunicação, superando outras soluções similares dentro da mesma instituição e se tornando uma referência em termos de eficiência no envio de dados.

/*LIBFT 🕮*/

Link para o Projeto

Uma biblioteca de funções em C para propósitos gerais de desenvolvimento, destacando-se pela sua eficiência e ausência de leaks de memória.

Descrição Completa

Funções Críticas: Desenvolvi uma biblioteca de funções em C para propósitos gerais de desenvolvimento, destacando-se pela sua eficiência e ausência de leaks de memória. A função printf() foi particularmente crucial, permitindo o debug de programas complexos em tempo de execução. Funções como split(), que divide blocos de dados em listas, e outras funções de validação foram essenciais para a construção de lógicas complexas e confiáveis.

Impacto: A LIBFT tornou-se uma ferramenta fundamental no meu kit de desenvolvimento, facilitando a criação de programas robustos e eficientes durante e após minha formação na 42SP.
