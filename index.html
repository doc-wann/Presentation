Rede de Web Scrappers para Estudo de Mercado

Como desenvolvedor lead, conduzi o desenvolvimento de uma robusta rede de web scrappers, projetada para coletar e analisar dados de mercado em larga escala, com uma abordagem que prioriza a discrição e a eficiência.

Especificações Técnicas:

  • Linguagens e Frameworks: C#, Next.js
  • Mascaramento de IP: Proxy rotativo, VPN, AWS
  • Controle de Detecção: Remoção de indicadores de automação, headers fictícios, comportamento humano simulado

Impacto e Utilidade:

Este projeto foi crucial para a empresa parceira, oferecendo uma ferramenta que permitiu a análise precisa e em tempo real de tendências de mercado.

Assistente de Mesas de RPG - D&D 5e

Uma ferramenta essencial para jogadores de Dungeons & Dragons 5ª Edição, com um foco em simplicidade e eficiência.

Especificações Técnicas:

  • Stack de Desenvolvimento: React, CSS
  • Localização: PT-BR nativo com suporte a outras línguas
  • Autosuficiência: Pode ser executado em qualquer máquina sem complicações adicionais

Impacto e Utilidade:

A ferramenta tornou-se uma referência na comunidade de jogadores, principalmente pela sua acessibilidade e foco em usabilidade.

Data-Parser para Grandes Volumes de Dados

Uma solução avançada para o processamento e validação de grandes volumes de dados, utilizando técnicas de inteligência artificial para garantir a precisão e uniformidade dos dados.

Especificações Técnicas:

  • Linguagens e Ferramentas: C#, Python
  • Validação de Dados com IA: LLMs com fine-tuning em bancos de dados
  • Otimização e Performance: Otimizado para PostgreSQL, técnicas de sorting, CleanCode

Impacto e Utilidade:

A ferramenta foi essencial para a empresa parceira, permitindo a correção e uniformização de grandes bancos de dados e identificação de oportunidades de mercado.
